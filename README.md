# windows-shortcuts
a handy visual guide of windows shortcuts
works with windows XP, 7, 8, 10, 11(?)

| Shortcut | Description    |
| :-----:  | :---:          | 
| Win + D  | Show Desktop   |
| Win + L  | Lock Computer  | 
| Ctrl + A | Select All     | 
| Ctrl + C | Copy           | 
| Ctrl + V | Paste          | 
| Win + B  | Select Taskbar |
